# Meuprojeto

Este é um projeto simples de **login** utilizando **HTML**, **CSS** e **JavaScript**, com integração com o **Supabase** para autenticação de usuários. O projeto permite a realização de login com e-mail e senha, cadastro de novos usuários, e visualização de uma lista de usuários após a autenticação.

## Funcionalidades

- **Página de Login**: Formulário para login com e-mail e senha.
- **Página de Cadastro**: Link para criar uma nova conta.
- **Página de Lista de Usuários**: Exibe todos os usuários cadastrados no banco de dados.

## Tecnologias Utilizadas

- **HTML** - Estruturação do conteúdo da página.
- **CSS** - Estilização visual das páginas (com um estilo neon).
- **JavaScript** - Interatividade e integração com o Supabase para autenticação e manipulação de dados.
- **Supabase** - Backend para autenticação e armazenamento de dados dos usuários.
